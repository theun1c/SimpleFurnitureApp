# Разработка системы управления продукцией мебельной компании

## Описание проекта
Необходимо разработать программный модуль на Avalonia UI (C#) для работы с продукцией компании, занимающейся изготовлением мебели. Система должна обеспечивать хранение, отображение, поиск, фильтрацию, сортировку, добавление, редактирование и удаление данных о продукции.

## Требования к оформлению
- **Основной фон:** #FFFFFF (белый)
- **Дополнительный фон:** #D2DFFF
- **Цвет акцента (для действий):** #355CBD
- **Иконка приложения и логотип компании** должны быть установлены на главной форме.

## Часть 1: Основной функционал отображения и фильтрации

### Отображение продукции
Каждый продукт отображается в виде карточки со следующей информацией:
- Тип
- Наименование продукта
- Артикул
- Время изготовления
- Минимальная стоимость для партнера
- Основной материал

**Примечание:** Время изготовления — целое неотрицательное число, складывается из времени нахождения в каждом цехе.

### Функционал главного окна
1. **Поиск:** Поисковая строка для поиска продуктов по наименованию. Если результаты не найдены — отображать соответствующее сообщение.
2. **Фильтрация:** Выпадающий список для фильтрации по типу продукта. Первый элемент списка — «Все» (сброс фильтра).
3. **Сортировка:** Сортировка по возрастанию/убыванию по полю «Минимальная стоимость для партнера».
4. **Режим работы:** Поиск, фильтрация и сортировка должны работать одновременно и в реальном времени.

## Часть 2: Управление данными и интерфейс

### Добавление/редактирование продукции
- **Кнопка «Создать»** на главной форме открывает форму добавления новой продукции.
- **Кнопка редактирования** на карточке продукта открывает форму редактирования.
- После сохранения происходит автоматическое обновление списка на главной форме.

### Поля формы добавления/редактирования (обязательные):
- Наименование
- Тип продукции (выпадающий список)
- Артикул
- Минимальная стоимость для партнера (целое неотрицательное число)
- Основной материал (выпадающий список)

### Удаление продукции
- Кнопка удаления на карточке продукта.
- Перед удалением запрашивается подтверждение пользователя.
